# ableton-companion

## Project setup
Install Vue3 then
```
npm install
```

### Compiles and hot-reloads for development
```
npm run electron:serve
```

### Compiles and minifies for production
Set the version of node to 16.0.0
```
nvm install 16.0.0
nvm use 16.0.0
```
Build for windows
```
npm run electron:build:win
```
Build for mac
```
npm run electron:build:mac
```
